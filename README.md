Example Chrome Extension to fetch the latest data on covid of UK
=======

# Developed using
* HTML, CSS and JavaScript
* [Bootstrap 5](https://getbootstrap.com/docs/5.0/getting-started/introduction/)

# How to Deploy?
* Goto Chrome -->Extensions --> Change to Developer Mode --> Load Unpacked --> Select the project --> Pin to Chrome if needed.
